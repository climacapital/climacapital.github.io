# Clima Company Website

## Stack
- Hugo
- TailwindCSS
- AlpineJS
- DatoCMS
- Netlify

## Install Hugo

Run from inside the root folder of the project:

```
brew install hugo
```

## Install Packages

Run from inside the root folder of the project:

```
npm install
```

## Run server

Run from inside the root folder of the project:

```
hugo server --disableFastRender
```

## Fetch content from DatoCMS

Run from inside the root folder of the project:

```
npm run fetch:dato
```
